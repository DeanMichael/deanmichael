# Personal Website

A modern, responsive personal website with parallax scrolling effects, built for GitHub Pages.

## Features

- Modern, clean design
- Parallax scrolling effects
- Responsive layout
- Smooth animations
- Mobile-friendly navigation
- Social media integration

## Setup Instructions

1. Fork this repository
2. Replace the content in `index.html` with your personal information
3. Add your own images to the `images` directory:
   - `hero-bg.jpg` for the home section background
   - `experience-bg.jpg` for the experience section background
   - `contact-bg.jpg` for the contact section background
4. Update the social media links in the contact section
5. Enable GitHub Pages in your repository settings:
   - Go to Settings > Pages
   - Select `main` branch as the source
   - Save the changes

## Customization

### Images
Place your background images in the `images` directory:
- Recommended size: 1920x1080px or larger
- Format: JPG or PNG
- Optimize images for web to ensure fast loading

### Colors
To change the color scheme, modify the following in `css/style.css`:
- Primary color: `#3498db`
- Secondary color: `#2c3e50`
- Background colors
- Text colors

### Content
Edit `index.html` to:
- Update your name and title
- Modify section content
- Add/remove sections as needed
- Customize social media links
